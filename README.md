# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/e4d9bf54-b2ea-4263-9688-3b54c9d5b2bf

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the app:
   ```bash
   npm run dev
   ```
3. Configure your Nano Banana API Key:
   - Open the app in your browser
   - Click the **Settings** icon (⚙️) in the header
   - Paste your Nano Banana/Gemini API key
   - Click **"Save Key"**

**Get Your API Key:**
- Visit [Google AI Studio](https://ai.google.dev/gemini-api/docs/billing)
- Follow the [API Key Setup Guide](./API_KEY_SETUP.md) for detailed instructions

> The API key is stored securely in your browser's localStorage and is never sent to any server except Google's Gemini API.
